Overview

New York Life Investments needed to improve the usability, consistency, and scalability of their website — a large ecosystem with dozens of live pages maintained by multiple teams. I led UX/UI enhancements across key areas, including interactive tools, contact forms, layout templates, and documentation within the shared design system.

Goals
  • Create reusable, accessible components within the design system
  • Update key page templates for consistency and responsiveness
  • Improve user experience for forms, calculators, and content layouts
  • Support implementation using AEM modules and QA updates
Key UX/UI Enhancements

Contact Form Overhaul

  • Improved layout and spacing for clarity
  • Added confirmation states and validation messages
  • Optimized for accessibility (labels, focus states)

Interactive Calculator

  • Designed user flow and UI for calculator tool
  • Built inputs, sliders, and dynamic result states
  • Collaborated with dev team for QA in AEM

Page Template & Design System Updates

Revised layouts for hero sections, content blocks, and CTAs using grid-based designs and system tokens. Documented new variants of buttons, cards, and other modules to support faster rollout by marketing teams.

Process

I followed an agile workflow: gathering requests, prototyping improvements, testing updates in staging, and collaborating directly with developers and content authors in AEM. Documentation ensured consistency across all modules and implementations.

Outcomes
  • Enhanced user experience across multiple high-traffic pages
  • Reduced implementation time for new pages via reusable components
  • Improved site accessibility and visual consistency
  • Extended internal design system documentation and usage
